🚀 JavaScript - Exercices basiques

Plongez dans l'univers du code avec des exercices interactifs

📋 Instructions

Ce projet contient 7 exercices JavaScript basiques. Pour tester chaque exercice :

  1. Sélectionnez le script à exécuter dans le menu déroulant
  2. Cliquez sur "Charger le script"
  3. Ouvrez la console de votre navigateur (F12 → Console)
  4. Suivez les instructions affichées dans la console

💻 Comment ouvrir la console :

  • Linux/Windows : F12 puis onglet "Console"
  • Mac : Alt + Cmd + I

Les résultats des exercices s'afficheront dans la console du navigateur.
